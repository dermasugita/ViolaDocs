Viola 1.0.1 Release Notes

Google Group for Viola-SV Users

To facilite discussion between users and developers, we’ve set up a google group for Viola-SV users! You can enter our group from here. Feel free to join us.

Fixed Features

  • Fixed feature in viola.read_vcf where patient_name were not reflected in the Vcf class.

  • Fixed feature in viola.read_bedpe where the confidence intervals were shifted by 1 to the upstream side (5’ end).
