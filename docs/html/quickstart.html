Quick Start¶

This is a short examples how to analyse vcf files.

When you read this page, you can

  • Import PySgt package to your script.

  • Read vcf file to create PySgt core object.

  • Convert the vcf file into a bedpe file, with any features you want such as INFO fields, and FORMAT fields.

  • Filter the SVs by any features, including genomic positions, INFO fields, and FORMAT fields.

Import PySgt to your script¶

How to import PySgt package is as follow:

In [1]: import sgt

Read the vcf file¶

In order to use PySgt’s features, you must first import a vcf file to create an SgtCore object.

In [2]: sgt_object = sgt.read_vcf('tutorial.vcf')

Now you’re ready to perform a number of functions that the PySgt package has.

Viewing the contents of SgtCore object¶

You may want to view the content of SgtCore objects after creating them. To do so, you have three options.

  1. Simply print the object.

In [3]: print(sgt_object)
INFO=imprecise,svtype,svlen,end,cipos,ciend,cigar,mateid,event,homlen,homseq,svinslen,svinsseq,left_svinsseq,right_svinsseq,contig,bnd_depth,mate_bnd_depth,somatic,somaticscore,junction_somaticscore,inv3,inv5
      id                          pos strand  qual svtype
0  test1  chr1:82550461-chr1:82554225     +-  None    DEL
1  test2  chr1:22814216-chr1:92581131     ++  None    INV
2  test3  chr1:60567906-chr1:60675940     +-  None    DEL
3  test4  chr1:69583190-chr1:69590947     +-  None    DEL

  1. Generate bedpe like pandas.DataFrame.

In [4]: sgt_bedpe_like = sgt_object.to_bedpe_like()

In [5]: print(sgt_bedpe_like)
  chrom1    start1      end1 chrom2    start2      end2   name score strand1 strand2
0   chr1  82550461  82550462   chr1  82554225  82554226  test1  None       +       -
1   chr1  22814216  22814217   chr1  92581131  92581132  test2  None       +       +
2   chr1  60567906  60567907   chr1  60675940  60675941  test3  None       +       -
3   chr1  69583190  69583191   chr1  69590947  69590948  test4  None       +       -

The way to add INFO, FILTER, and FORMAT to this bedpe-like DataFrame is explained here (no link for now).

  1. Use the get_table() method to get individual tables composing the SgtCore object.
