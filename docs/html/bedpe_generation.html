Flexible BEDPE generation¶

The first step is to create an SgtCore object.

In [1]: import sgt

In [2]: sgt_object = sgt.read_vcf('tutorial.vcf')

Then use to_bedpe_like method to generate bedpe-formatted pandas DataFrame. The most simple code is like:

In [3]: sgt_bedpe_like = sgt_object.to_bedpe_like()

If you want to add “SVLEN” and “CIPOS” fields, run as follows:

In [5]: sgt_bedpe_like_with_info = sgt_object.to_bedpe_like(custom_infonames=['svlen', 'cipos'])
