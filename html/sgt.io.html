read_vcf(filepath_or_buffer, variant_callser='manta')¶

Read vcf file of SV and return SgtCore object.

Parameters

  • filepath_or_buffer (str or StringIO) – String path to the vcf file. StringIO is also acceptable. (Wether URL is acceptable or not hasn’t been tested.) (Acceptable types should be extended in the future)

  • variant_caller (str) – Let this function know which SV caller was used to create vcf file. Only “manta” is supported for now.

Returns

Return type

A SgtCore object
